Angular Pipe

Pipes are used to convert data as per need and display on browser

Some default pipes provided by angular as below

Command to create custom pipe ng g p pipeName

Uppercase Pipe

{{title | uppercase}}

Lowercase Pipe

{{title | lowercase}}

Date pipe

{{todaydate }} - Original Date
{{todaydate | date:'d/M/y'}} - format d/M/y
{{todaydate | date:'dd-MMMM'}} - format dd-MMM
{{todaydate | date:'dd-MMM-yy'}} - format dd-MMM-yy
{{todaydate | date:'dd-MMM-yyyy'}} - format dd-MMM-yyyy
{{todaydate | date:'shortTime'}} - short time

Currency Pipe

{{6589.23 | currency:"USD"}}

Decimal Pipe

{{ 454.78787814 | number: '3.3-3' }} // 3 is for main integer, 4 -4 are for integers to be displayed.

Percent Pipe

{{00.54565 | percent}}

Json Pipe

{{ jsonval | json }}

Slice Pipe

{{months | slice:0:6}} // here 2 and 6 refers to the start and the end index
